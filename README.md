# Vulnerability Scanner

## Task Description
Built a Vulnerability Scanner web application that simulates scanning websites for common security vulnerabilities and generates a simple security report.

## Features
- Website URL input
- Simulated vulnerability scanning
- HTTPS security detection
- Open port analysis
- Outdated software detection
- Security score generation
- Animated scan effect
- Cybersecurity-themed hacker UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
The application analyzes the entered website URL and simulates:
- HTTPS verification
- Open port detection
- Software version analysis

A security score is then generated based on the scan results.

## Output
The scanner displays:
- Security report
- Detected vulnerabilities
- Security score
- Threat indications

## Note
This project is created for educational purposes and demonstrates basic concepts of vulnerability assessment and cybersecurity analysis.
